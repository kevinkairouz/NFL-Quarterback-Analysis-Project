NFL QB Statistical Analysis
A comprehensive Python data visualization project tracking touchdowns, completion percentages, and interceptions of four elite NFL quarterbacks over a five-year period (2020-2024).
Overview
This project analyzes and visualizes the passing accuracy trends of four prominent NFL quarterbacks:

Patrick Mahomes (Kansas City Chiefs)
Lamar Jackson (Baltimore Ravens)
Josh Allen (Buffalo Bills)
Aaron Rodgers (Green Bay Packers / New York Jets)

Features

Multiple line graph visualizations with distinct styling for each quarterback
Year-by-year tracking of three key metrics (2020-2024):

Touchdowns (TDs): Total passing touchdowns per season
Completion Percentage (CP): Accuracy of completed passes
Interceptions (INTs): Turnovers thrown per season


Custom markers and colors for easy player identification
Grid lines for improved readability
Separate visualizations for each statistical category

Data Source
Statistical data includes touchdowns, completion percentages, and interceptions for each quarterback's season performance from 2020 through 2024.
Visualization Details

Mahomes: Red line with circle markers
Jackson: Purple line with star markers
Allen: Blue line with triangle markers
Rodgers: Green line with circle markers

All lines feature dashed styling for visual distinction.
Key Insights
The visualizations allow for easy comparison of:

Individual quarterback consistency and performance trends over time
Touchdown production across seasons
Passing accuracy (completion percentage) trends
Ball security and interception rates
Relative performance between elite quarterbacks across multiple metrics
