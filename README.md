# NFL QB Analysis Projet

A Python data visualization project tracking the completion percentages, touchdowns, and wins of four elite NFL quarterbacks.

## Overview

This project analyzes and visualizes the passing accuracy trends of four prominent NFL quarterbacks:
- **Patrick Mahomes** (Kansas City Chiefs)
- **Lamar Jackson** (Baltimore Ravens)
- **Josh Allen** (Buffalo Bills)
- **Aaron Rodgers** (Green Bay Packers / New York Jets)

## Features

- Line graph visualization with distinct styling for each quarterback
- Year-by-year completion percentage tracking, and touchdown tracking (2020-2024)
- Custom markers and colors for easy player identification
- Grid lines for improved readability
- All time Wins
  

## Data Source

Completion percentage data represents each quarterback's season performance from 2020 through 2024.

## Visualization Details

- **Mahomes**: Red line with circle markers
- **Jackson**: Purple line with star markers
- **Allen**: Blue line with triangle markers
- **Rodgers**: Green line with circle markers

All lines feature dashed styling for visual distinction.

## Key Insights

The visualization allows for easy comparison of:
- Individual quarterback consistency over time
- Performance trends and trajectories
- Relative accuracy between elite quarterbacks
